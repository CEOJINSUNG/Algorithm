  1. 매일 한개 이상의 Algorithm 문제를 풀고 인증하기(못 풀었을 경우 벌금)
  2. 백준 알고리즘 '단계별로 풀기'를 올해 다 마무리하기(https://www.acmicpc.net/step)
  
#### 공부과정
